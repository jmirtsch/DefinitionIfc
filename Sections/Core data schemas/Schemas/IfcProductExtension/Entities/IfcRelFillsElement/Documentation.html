IfcRelFillsElement is an objectified relationship between an opening element and an element that fills (or partially fills) the opening element. It is an one-to-one relationship.

NOTE  View definitions or implementer agreements may restrict an opening to be filled by one filling element only.

As shown in Figure 161, the insertion of a door into a wall is represented by two separate relationships. First the door opening is created within the wall by IfcWall(StandardCase) <-- IfcRelVoidsElement --> IfcOpeningElement, then the door is inserted within the opening by IfcOpeningElement <-- IfcRelFillsElement --> IfcDoor.
